## README
 # Mosquito Data Analysis

This project generates a random dataset of mosquito species counts over a period of three years and creates contingency tables for analysis. The data is exported to an Excel workbook.

## Steps:
1. **Generate Dataset**: 
   - Four mosquito species (`Aedes`, `Anopheles`, `Culex`, `Mansonia`) are randomly generated with counts and dates spanning from `2021-01-01` to `2023-12-31`.
   - The dataset includes 1000 observations.
   
2. **Create Contingency Table**: 
   - A contingency table is created from the dataset, showing the distribution of species across the dates.

3. **Export to Excel**:
   - The dataset and the contingency table are saved in an Excel workbook (`Mosquito_Data.xlsx`).

Refer to the R script for generating the data and creating the Excel file.
